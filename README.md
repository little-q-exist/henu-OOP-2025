## 使用/Usage

* 在src\main\Main.java运行程序
* 如果使用vscode打开代码，需要把fishGame作为项目根目录 
* 如果遇到路径问题，请把src文件夹作为Java源代码路径！！
* 按照老师的要求，去除了体积较大的 **"皇家萌卫.wav"**，请老师在添加音乐的时候，把音乐文件放在以下路径：
#### src\res\皇家萌卫.wav
(音乐是包含在仓库内的)
