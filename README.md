## 下载/Installation

Clone the repository:
```bash
git clone https://github.com/little-q-exist/henu-OOP-2025.git
```

## 使用/Usage

* 在src\main\Main.java运行程序
* 如果使用vscode打开代码，需要把fishGame作为项目根目录 
* 如果遇到路径问题，请把src文件夹作为Java源代码路径！！
### 功能
1. 方向键控制移动
2. Q键进入调试模式
